### 1. 文字の装飾・基本構造

文章の強調や構造化に使う基本的な記法です。

| **項目**    | **記法（書き方）**                         | **表示イメージ / 備考**                                            |
| --------- | ----------------------------------- | ---------------------------------------------------------- |
| **見出し**   | `# 見出し1`<br>`## 見出し2`<br>`### 見出し3` | `#` の数（1〜6個）で見出しの大きさを変えます。<br><br>**※`#` の後に半角スペースが必要です。** |
| **太字**    | `**強調したい文字**`                       | **強調したい文字**                                                |
| **斜体**    | `*斜体にする文字*`                         | *斜体にする文字*                                                  |
| **取り消し線** | `~~削除する文字~~`                        | ~~削除する文字~~                                                 |
| **水平線**   | `---` または `***`                     | 区切り線が表示されます。                                               |
| **改行**    | 行末に半角スペース2つまたは空行を挟む                 | （Markdownの仕様上、単なる改行は無視されることが多いです）                          |

### 2. リスト・リンク・画像

箇条書きやリンクの埋め込みに使用します。

| **項目**      | **記法（書き方）**                     | **表示イメージ / 備考**               |
| ----------- | ------------------------------- | ----------------------------- |
| **箇条書き**    | `- リストA`<br>`- リストB`            | 記号は `-` `*` `+` のどれでもOKです。    |
| **番号付きリスト** | `1. 手順1`<br>`2. 手順2`            | 自動的にインデントされます。                |
| **チェックリスト** | `- [ ] 未完了タスク`<br>`- [x] 完了タスク` | プレビュー画面でチェックボックスになります。        |
| **リンク**     | `[表示テキスト](URL)`                 | [表示テキスト](https://google.com/) |
| **画像埋め込み**  | `![代替テキスト](画像URL)`              | 画像が表示されます（リンク記法の頭に`!`をつける）。   |

### 3. コード・引用・表（テーブル）

少し応用的な記法ですが、ドキュメント作成で非常によく使われます。

#### コードの埋め込み

- **インライン（文中）:** バッククォート(`)で囲みます。
   - 書き方: `` `code` ``
   - 表示: `code`
- **コードブロック:** バッククォート3つ(```)で上下を囲みます。言語名を指定するとシンタックスハイライト（色付け）が効きます。   

Markdown

````
```python
print("Hello, World!")
```
````

#### 引用

行頭に `>` をつけます。`>>` と重ねることで階層化できます。

> これは引用です。
> 
> > これは入れ子の引用です。

#### 表（テーブル）

パイプ `|` とハイフン `-` を組み合わせて作成します。コロン `:` の位置で文字寄せを指定できます。

**書き方:**

Markdown

```
| 左揃え | 中央揃え | 右揃え |
| :--- | :---: | ---: |
| Item 1 | Item 2 | Item 3 |
| Apple | Banana | Orange |
```

**表示結果:**

| **左揃え** | **中央揃え** | **右揃え** |
| ------- | -------- | ------- |
| Item 1  | Item 2   | Item 3  |
| Apple   | Banana   | Orange  |

### 💡 知っておくと便利なポイント

1. エスケープ処理:
    
    Markdownの記号（# や * など）をそのまま文字として表示したい場合は、直前にバックスラッシュ \ を置きます。
    
    - 例: `\# ハッシュタグ` → **# ハッシュタグ** （見出しにならない）  
1. 数式の挿入（LaTeX）:    
    多くのMarkdownエディタ（Qiita, Zenn, Notion, GitHub等）では、ドルマーク $ で囲むことで数式を書けます。
    
    - 書き方: `$E = mc^2$`
    - 表示: $E = mc^2$
