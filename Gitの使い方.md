| 場所     | 定義                                                                                                                               |
| ------ | -------------------------------------------------------------------------------------------------------------------------------- |
| リポジトリ  | ファイルや変更履歴を保存しておくデータベース。リポジトリには２種類ある。<br>・ローカルリポジトリ（ローカル上）：ローカル環境で作業を行う際に使う。<br>・リモートリポジトリ（ネットワーク上）：ほかのユーザーとファイルやへhン港履歴を共有する際に使う。 |
| インデックス | リポジトリ（保存場所）とワークツリー（作業場所）の間の中間領域。ローカルで作業したファイルをリポジトリに保存するには、インデックスにいったん置く必要がある。                                                   |
| ワークツリー | ユーザーが編集している作用中のディレクトリのこと。                                                                                                        |
